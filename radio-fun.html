Networks and links

These Networks often have many Connections Methods. Some also use Zello and TeamSpeak for network radio style operation.

Australia and New Zealand Network (ANZEL)

A large network with nets and linking across IRLP, AllStar, DMR, and EchoLink connections.

International Radio Network

Promotes amateur radio worldwide, welcoming licensed and non-licensed users. Access may include TeamSpeak, Zello, DMR, AllStar, System Fusion, and & More.

International Radio Network website

ZMR Radio Network

Network radio and ZMR over Zello can complement ham radio as another operating mode. Like other linked systems, it relies on internet connectivity.

ZMR Radio website

Network Radio

Zello-based network radio channels for licensed and unlicensed users. They cater for everyone, plus a shop dedicated to Network Radios.

Network Radios website
